# PersonalFinanceManager
A lightweight finance tracking system that helps users understand, organize, and optimize their daily expenses. It provides a clear view of income, spending categories, savings goals, and monthly trends. The system assists in budgeting, visualizing cash flow, and making smarter financial decisions through clean dashboards and automated summaries.
